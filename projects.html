PROJECTS

Here is a list with description of the projects I've worked in. My Android experience is with the music apps, iRadio, TapRadio and ideasmusik. Also for iRadio, I've developed code for the server side using C#.NET. You can see also my most important web developments, using HTML, CSS, jQuery and ASP.NET MVC 3. Finally, I show my Masters Degree project and the most important project I did in college.

Projects working for SMI

iRadio: iRadio is a music streaming mobile application developed for America Móvil, the major mobile service provider in Mexico and Latin America. It is available for Android, iOS, Blackberry, J2ME and Windows Phone. You can listen to over 100 radio stations and create custom stations based on your favorite songs, albums or artists. I have been involved in developing new features and giving maintenance to the API, using C#.NET. Also, I have worked on J2ME and Blackberry versions, from fixing bugs to implementing some minor new features. The most important work of all, has been in the Android version. I have implemented new features and fixed bugs in the old version. Right now, I am in charge of the completely new version for Android, and I have been involved in the software and graphical design, implementation and bug fixing of this new version. You can see an image gallery of iRadio old version, and some screenshots from the new version. You can download the apk file here.

ideasmusik: ideasmusik is a music store and on-demand streaming application developed for America Móvil too. You can buy songs, search and listen for your favorite songs, albums and artists, add songs to a favorites list and download the songs you have bought. In the new version that is under development, you can buy subscription plans too. I developed the entire application for Android. You can see here an image gallery of ideasmusik. This app is only available in Mexico and will be available in other Latin American countries soon. SMI developed the Android version from version 1.0 to version 2.5. You can download the apk file here.

Mobzilla Radio: Mobzilla Radio is a radio station streaming app for Android. It uses Dar.Fm API. I started working on this project when it was half way done. I have developed API calls, UI, and app logic. With this app, you can listen to local radio staions, you can see a list of top songs for different genres and start listening to that song on a radio station from all over the world. Also you can search for your favorite stations, other world stations, or stations that are currently playing the music that you want. You can download this app here. I also created a small API to let the user see featured stations in the app.

Mobzilla.com: Mobzilla.com is a website where you can find information about top latin artists. You can search for your favorite artist, see their songs, albums, Twitter account, videos and biography. I created this website using ASP.NET MVC 4 and C# .NET, using SQL 2008. This website has responsive web design. You can visit the website here.

Independent projects

Music Multimedia Store: Website and web application that lets any music band or artist to manage their content (songs, albums, pictures, videos). This content will be displayed on their website and mobile apps. For this project I used ASP.NET MVC 3 and SQL Server 2008. The Vendetta Band website and their Android app were done using Music Multimedia API. You can download here Vendetta band Android app. This project is still under development.

Vendetta Band WebSite: I developed the website for Mexican rock band Vendetta. It was made using ASP.NET MVC 3 and responsive web design. This website uses the Music Multimedia Store API and database. This project is under development, it is a music content manager for different music bands, where they can manage all their content and have this content shown in a mobile app or website. This website is the first project created using this service. This website also uses Google Analytics.

Embolización de miomas uterinos website: This is the website for interventional radiologist Dr. Guadalupe Guerrero. She uses this technique to remove myomas from women's uterus. This is a simple website developed using HTML, CSS and jQuery. It also uses Google Analytics.

Sistemas de Control Industrial y Automatización website: This is the website for mexican enterprise Sistemas de Control Industrial y Automatización. It is an enterprise that develops automatic control for industries like paper, cardboar and rubber. This was the first website that I developed. It was created using HTML, CSS and jQuery.

Academic projects

The Reionization and Transients Infrared/Optical Project (RATIR): This was the project I worked on in my Masters Degree. I proposed several mechanical designs for the mechanic support for the dichroics that this instrument uses. Then I analysed each design and with Finite Element Method I simulated the deformations that the support generates in the dichroic. Also, using CNC I manufactured a model to take measurements in the laboratory. My project was selected to be presented at the International Symposium on Microwave and Optical Technology in Prague, Czech Republic on June 2011.

Nautilus Rescue Robot: I was part of the electronic system development team, and took part in the design and implementation of the motors and sensors controls for the robot. This robot later evolved to become Robot Kauil. We presented this project at the IEEE Latin American Symposium (LARS 2007) in Monterrey, Mexico.
